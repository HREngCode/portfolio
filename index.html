A Little About Me!

I've been technically driven for as long as I can remember, but my interest in web development started about 20 years ago. I ran across a site that had a tutorial for building a website in HTML and I found myself working through the exercises at my computer on my breaks. I spent most of my time learning the functionality of the tags and the commands until our company was sold.

I later developed a knack for looking at code and trying to figure what the syntax was doing. No matter what language the syntax was coded in, I would make it a personal goal to interpret the code and try to replicate the command or functionality. My hope was to one day perform coding for a living.

After spending 12 years in Human Resources, I wanted to make that dream a reality so I enrolled in a Full Stack Web Developer Boot Camp. After graduating, I still felt called to learn more so I enrolled in another Full-Stack Boot Camp that had more of an emphasis on Front End Technologies. Not only has it improved my skills, but it has boosted my confidence as well.
